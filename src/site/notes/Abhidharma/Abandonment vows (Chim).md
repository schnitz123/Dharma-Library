བར་ཆད་མེད་པའི་ལམ་དགུ་དག་ཏུ་སྐྱེས་པའི་བསམ་གཏན་དང་ཟག་མེད་ཀྱི་སྡོམ་པ་དེ་གཉིས་ནི་སྤོང་བའི་སྡོམ་པ་ཞེས་བྱའོ།
The concentration and uncontaminated vows that are generated as the nine uninterrupted paths are called abandonment vows.

བར་ཆད་མེད་པའི་ལམ་ཐམས་ཅད་དམ་ཞེ་ན། 
བསམ་གཏན་དང་པོའི་ཉེར་བསྡོགས་མི་ལྕོགས་པ་མེད་པ་ལ་ཡིན་ཏེ། འཆལ་བའི་ཚུལ་ཁྲིམས་དང་དེ་ཀུན་ནས་སློང་བའི་ཉོན་མོངས་རྣམས་སྤོང་བའི་ཕྱིར་རོ།
Does that apply to all uninterrupted paths?
It is those of the "Not Unable" preparation of the first concentration because of abandoning faulty ethics and the afflictions motivating those.

Therefore, there are four possibilities: དེའི་ཕྱིར། ་་་མུ་བཞི་དང་། 
- Is a concentration vow but is not an abandonment vow: Contaminated concentration vows excluding uninterrupted path of the "Not Unable"
  བསམ་གཏན་གྱི་སྡོམ་པ་ཡིན་ལ། སྤོང་བའི་སྡོམ་པ་མ་ཡིན་པ་ནི། མི་ལྕོགས་པ་མེད་པའི་བར་ཆད་མེད་པའི་ལམ་མ་གཏོགས་པའི་ཟག་བཅས་ཀྱི་བསམ་གཏན་གྱི་སྡོམ་པའོ།
- Is an abandonment vow but is not a concentration vow: An uncontaminated vow that is an uninterrupted path of the preparations
  སྤོང་བའི་སྡོམ་པ་ཡིན་ལ་བསམ་གཏན་གྱི་སྡོམ་པ་མ་ཡིན་པ་ནི། ཉེར་བསྡོགས་ཀྱི་བར་ཆད་མེད་པའི་ལམ་གྱི་ཟག་མེད་ཀྱི་སྡོམ་པའོ།
- Both: A concentration vow that is an uninterrupted path of the "Not Unable"
  གཉིས་ཀ་ཡིན་པ་ནི། ཉེར་བསྡོགས་མི་ལྕོགས་མེད་པའི་བར་ཆད་མེད་པའི་ལམ་གྱི་བསམ་གཏན་གྱི་སྡོམ་པའོ།
- Neither: An uncontaminated vow not included in those and not included in a path of the "Not Unable"
  གཉིས་ཀ་མ་ཡིན་པ་ནི། དེ་ལས་མ་གཏོགས་པ་མི་ལྕོགས་པ་མེད་པའི་ལམ་མ་གཏོགས་པའི་ཟག་མེད་ཀྱི་སྡོམ་པའོ།

དེ་བཞིན་དུ་ཟག་མེད་ཀྱི་སྡོམ་པ་ཡིན་ལ་སྤོང་བའི་སྡོམ་པ་མ་ཡིན་པ་ལ་སོགས་པའི་མུ་བཞིའང་ཤེས་པར་བྱའོ། །
Similarly, one should also understand the four possibilities for that which is an uncontaminated vow and an abandonment vow, and so forth.
