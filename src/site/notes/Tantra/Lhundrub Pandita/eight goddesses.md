**The four female gatekeepers:** སྒོ་མ་བཞི། 
1. Aṅkuśī ལྕགས་ཀྱུ་མ།
2. Pāśadhārī ཞགས་པ་མ། 
3. Saṅkalā ལྕགས་སྒྲོག་མ། 
4. Ghaṇṭā དྲིལ་བུ་མ།

**The four goddesses of Cundavajrī and so forth**: སྐུལ་བྱེད་མ་སོགས་ལྷ་མོ་བཞི།
{ #95cccd}

1. Vasudhārā ནོར་རྒྱུན་མ། 
2. Cundavajrī སྐུལ་བྱེད་མ། 
3. Bhṛkuṭī ཁྲོ་གཉེར་མ་ཅན་མ། 
4. Āvalī ཕྲེང་བ་མ།
