**Five root winds:** རྩ་བའི་རླུང་ལྔ།
1. *Life-bearing wind* སྲོག་འཛིན་གྱི་རླུང་། prāṇāyāyu
   Heart, white, preserving life
2. *Downward-voiding wind* ཐུར་སེལ་གྱི་རླུང་། apāna
   Secret place, yellow, expelling waste products
3. *Equally-abiding wind* མཉམ་གནས་ཀྱི་རླུང་། samāna
   Navel, green/black, processing food etc.
4. *Upward-moving wind* གྱེན་དུ་རྒྱུ་བའི་རླུང་། udāna
   Throat, red, breathing.
5. *Pervading wind* ཁྱབ་བྱེད་ཀྱི་རླུང་། vyāna
   Entire body, blue, movement

**Five branch winds:** ཡན་ལག་གི་རླུང་ལྔ།
1. *Moving wind* རྒྱུ་བའི་རླུང་། / ཀླུའི་རླུང་། nāga
   Eye-faculty
2. *Strongly moving wind* རྣམ་པར་རྒྱུ་བའི་རླུང་། / རུས་སྦལ་གྱི་རླུང་། kūrma
   Ear-faculty
3. *Fully moving wind* ཡང་དག་པར་རྒྱུ་བའི་རླུང་། / རྩངས་པའི་རླུང་། kṛkara
   Nose-faculty
4. *Intensely moving wind* རབ་ཏུ་རྒྱུ་བའི་རླུང་། / ལྷ་སྦྱིན་གྱི་རླུང་། devadatta
   Tongue-faculty
5. *Definitively moving wind* ངེས་པར་རྒྱུ་བའི་རླུང་། / ནོར་རྒྱལ་གྱི་རླུང་། dhanañjaya
   Body-faculty
