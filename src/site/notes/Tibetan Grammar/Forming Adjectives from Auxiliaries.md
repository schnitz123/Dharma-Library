These adjectives are mainly formed using the auxiliaries ཅན། ལྡན། བཅས། ཡོད། མེད།. 
These and would often be equivalent to an adjectival construction using the Sixth Case.

**Illustrations:**
- A tree with leaves ཤིང་ལོ་མ་ཅན། = ལོ་མ་ཅན་གྱི་ཤིང་། = ལོ་མ་ཡོད་པའི་ཤིང་།
- Contaminated ཟག་བཅས།
- Uncontaminated ཟག་མེད།
- Intelligent བློ་ལྡན། 
- Powerful དབང་ལྡན།
