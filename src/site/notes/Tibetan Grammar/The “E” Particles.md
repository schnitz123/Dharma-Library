The [[Tibetan Grammar/The Third Case – Agentive\|Third Case]] particle is suffix dependent:

| Suffix          | Marker      |
| --------------- | ----------- |
| ད། བ། ས།        | ཀྱིས།       |
| ག ང་།           | གིས།        |
| ན། མ། ར། ལ།     | གྱིས།       |
| འ། or no suffix | ཡིས། or -ས། |

Regarding the last of these, the ས། will “replace” an འ། suffix or be joined directly to the word:
- ནམ་མཁའ། + -ས། → ནམ་མཁས།
- བོད་པ། + -ས། → བོད་པས།

---
The [[Tibetan Grammar/The Sixth Case – Connective\|Sixth Case]] particle is suffix dependent and is affixed as follows:

| Suffix          | Marker      |
| --------------- | ----------- |
| ད། བ། ས།        | ཀྱི།        |
| ག ང་།           | གི།         |
| ན། མ། ར། ལ།     | གྱི།        |
| འ། or no suffix | ཡི། or -འི། |

Regarding the last of these, the འི། will “replace” an འ། suffix or be joined directly to the word:
- མཐའ། + -འི། → མཐའི།
- ཆུ། + -འི། → ཆུའི།
