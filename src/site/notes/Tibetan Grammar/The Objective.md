བྱ་བའི་ཡུལ་གང་ཞིག་ལ་བྱེད་པ་པོ་རང་གི་དོན་དུ་བྱེད་པ་ཉེ་བར་སྟོན་པའི་རྐྱེན། ལས་སུ་བྱ་བའི་སྒྲའི་གོ་དོན།
**Understood meaning:** A particle indicating an indirect object with respect to which the agent acts.

[[Tibetan Grammar/The Denyi\|The Denyi]] particle is an instance of this དེ་ཉིད་ཀྱི་སྒྲ་ཡང་འདིའི་བྱེ་བྲག་ཅིག་ཡིན།

**For example:**
- “Tashi goes to the east.” Indirect object: the east.
  བཀྲ་ཤིས་ཤར་ཕྱོགས་སུ་འགྲོ། འགྲོ་བར་བྱ་བའི་ཡུལ། = ཤར་ཕྱོགས།
- “The seeds were planted in the field.” Indirect object: the field.
  ས་བོན་ཞིང་ལ་བཏབ། གདབ་པར་བྱ་བའི་ཡུལ། = ཞིང་།
- “Losang looked at the flower.” Indirect object: the flower.
  བློ་བཟང་གིས་མེ་ཏོག་ལ་བལྟས། བལྟ་བར་བྱ་བའི་ཡུལ། = མེ་ཏོག

Another way to think about it, is that this case marks the destination for verbs that have some _directionality_ about them. As such, it acts as the complement to the [[Tibetan Grammar/The Fifth Case – Origin\|Fifth Case]] (which marks an origin) by marking a _destination_. In the above examples:
- The east is the place that will be gone _to_.
- The field is the place that will be planted _in_.
- The flower is the “place” that will be looked _at_.

For more on the idea of directionality, see: [[Tibetan Grammar/The Second Case – Objective#^1af50c\|directionality]]

This is reflected well in English, where we also mark these destinations (indirect objects) with “to”, “in”, “at”, and so forth.

**Difference between a direct object (ལས།) and an indirect object (བྱ་བའི་ཡུལ།)**
{ #99e499}


For verbs [[Tibetan Grammar/Where agent and object are different\|where agent and object are different]], the basis for the division is the action's basis (ལྟོས་གཞི།), which is then subdivided into:
1. *Those that are affected by the action* (བྱ་བའི་ཤུགས་རྐྱེན་ཐེབས་པ།)
   Considered the direct object (ལས།) and marked with the [[Tibetan Grammar/The First Case – Nominative\|First Case]]
2. *Those that are not affected by the action* (བྱ་བའི་ཤུགས་རྐྱེན་མི་ཐེབས་པ།)
   Considered the indirect object (བྱ་བའི་ཡུལ།) and marked with the Second Case
{ #046c02}


**For example:**
- "To cut wood." ཤིང་གཅོད།
  There is only one basis (ལྟོས་གཞི།), i.e. the wood, and it is considered the direct object (གཅོད་བྱ།) and not the indirect object (གཅོད་ཡུལ།) because it is affected by the action of cutting.
- “The seeds were planted in the field.” ས་བོན་ཞིང་ལ་བཏབ།
  There are two bases (ལྟོས་གཞི།): The direct object (གདབ་བྱ།) is the seed and the indirect object (གདབ་ཡུལ།) is the field.
- “Losang looked at the flower.” བློ་བཟང་གིས་མེ་ཏོག་ལ་བལྟས།
  There is only one basis (ལྟོས་གཞི།), i.e. the flower, and it is considered the indirect object (བལྟ་ཡུལ།) and not the direct object (བལྟ་བྱ།) because it is not affected by the action of being looked at.

This last type of verbs will not have a resultative like the examples of cutting wood (resultative: the wood having been cut) and planting the seed (resultative: the seed having been planted) because there is no definitive culmination and ceasing of the action; because the object is not affected by that action.
{ #ec5395}


There is a temptation (also in Tibetan) to leave these indirect objects in the [[Tibetan Grammar/The First Case – Nominative\|First Case]] since there is only one basis (ལྟོས་གཞི།), however it is considered more grammatically correct for them to be marked with the Second Case. Similarly in English, it is possible to say “to look at the sunset” or “to watch the sunset”, “to ride the horse” or “to ride on the horse”, etc.

---
**Summary**

The Second Case marks the indirect object or destination for verbs that have some directionality about them. 
- Despite the use of the word “objective”, it should not be confused for the [[Tibetan Grammar/The Fourth Case – Purpose\|Fourth Case]] marking a purpose. 
- Despite the use of the word “destination”, it should not be confused for the [[Tibetan Grammar/The Locative\|Seventh Case Locative]]. 

See: [[Tibetan Grammar/The Difference between the Second, Fourth, and Seventh Cases\|The Difference between the Second, Fourth, and Seventh Cases]]
