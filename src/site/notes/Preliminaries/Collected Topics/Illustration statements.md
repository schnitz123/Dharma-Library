*Response:* It follows that that is incorrect because a person who has ascertained golden-pot with valid cognition but has not ascertained pot with valid cognition does not exist.

ཡང་ཁ་ཅིག་ན་རེ། གསེར་བུམ་མཚན་གཞི། བུམ་པ་ཡིན་པར་མཚོན། ལྟོ་ལྡིར་བ་ཡིན་པ་ཞེས་པའི་མཚོན་སྦྱོར་འདི་མཚོན་སྦྱོར་རྣམ་དག་ཡིན་ཟེར་ན། 
*Someone says:* The illustration statement, "The illustration - golden-pot - is illustrated as being a pot, being a bulbous [thing]." is a correct illustration statement.

དེ་མི་འཐད་པར་ཐལ། ལྟོ་ལྡིར་བ་དེ་བུམ་པའི་མཚན་ཉིད་མ་ཡིན་པའི་ཕྱིར།
*Response:* It follows that that is incorrect because "bulbous" is not the definition of pot.

མ་གྲུབ་ན། ལྟོ་ལྡིར་བ་ཡིན་ན། བུམ་པ་ཡིན་དགོས་པར་ཐལ། ལྟོ་ལྡིར་བ་བུམ་པའི་མཚན་ཉིད་ཡིན་པའི་ཕྱིར། རྟགས་ཁས།
If reason not established: It follows that if it is bulbous, then it is necessarily a pot because "bulbous" is the definition of pot. You asserted the reason.

འདོད་ན། བུམ་པ་ཞབས་རྡོལ་བ་ཆོས་ཅན། བུམ་པ་ཡིན་པར་ཐལ། ལྟོ་ལྡིར་བ་ཡིན་པའི་ཕྱིར། 
If accepted: The subject - a pot with a leaky bottom - it follows that it is a pot because it is bulbous.

མ་གྲུབ་ན། དེ་ཆོས་ཅན། ལྟོ་ལྡིར་བ་ཡིན་པར་ཐལ། ལྟོ་ལྡིར་བ་ཡིན་པར་མངོན་སུམ་དུ་གྲུབ་པའི་ཕྱིར།
If reason not established: That subject, it follows that it is bulbous because it being bulbous is established by direct perception.

འདོད་ན། བུམ་པའི་མཚན་ཉིད་ཀྱི་ཟུར་དུ་ཆུ་སྐྱོར་བའི་དོན་བྱེད་ནུས་པ་ཞེས་སྨོས་པ་ལ་དགོས་པ་མེད་པར་ཐལ། 
བུམ་པ་ཞབས་རྡོལ་བུམ་པ་ཡིན་པའི་ཕྱིར། འདོད་མི་ནུས་སོ།
If accepted: It follows that there is no need to say "able to perform the function of carrying water" as part of the definition of pot because a pot with a leaky bottom is a pot. You cannot accept!

**Our own system** རང་ལུགས་བཞག་པ་ནི། 

གསེར་གྱི་ལྟོ་ལྡིར་ཞབས་ཞུམ་ཆུ་སྐྱོར་བའི་དོན་བྱེད་ནུས་པ་མཚན་གཞི། བུམ་པ་ཡིན་པར་མཚོན། 
ལྟོ་ལྡིར་ཞབས་ཞུམ་ཆུ་སྐྱོར་བའི་དོན་བྱེད་ནུས་པ་ཞེས་པའི་མཚོན་སྦྱོར་ལྟ་བུ་མཚོན་སྦྱོར་རྣམ་དག་ཡིན་ནོ།
The illustration statement, "The illustration - a golden, bulbous, flat-bottomed [thing] able to perform the function of carrying water - is illustrated as being a pot, being a bulbous, flat-bottomed [thing] able to perform the function of carrying water.", for example, is a correct illustration-statement.

དེ་ཡང་དོན་དེ་བློ་ལ་ཤར་ཡང༌། དེ་ལ་ཐ་སྙད་ཅི་འདོགས་མི་ཤེས་པའི་ངོར། དོན་འདི་འདྲའི་རིགས་ལ།
མིང་གི་ཐ་སྙད་འདི་ལྟར་ཟེར་ཞེས་བརྡ་སྦྱོར་ཞིང་མཚོན་པར་བྱེད་དགོས་ལ།
It is for those for whom the meaning has dawned to awareness but do not know what term to designate it with that it is necessary to apply the term and illustrate it saying, "These type of things are called this name."

དཔེར་ན་གདུང་འདེགས་ཀྱི་དོན་བྱེད་ནུས་པ་བློ་ལ་ངེས་ཟིན་ཀྱང༌། དེ་ལ་ཀ་བ་ཞེས་པའི་ཐ་སྙད་སྦྱོར་མི་ཤེས་པའི་གང་ཟག་ལ། གདུང་འདེགས་ཀྱི་དོན་བྱེད་ནུས་པ། 
ཀ་བའི་མཚན་ཉིད་དམ་དོན་ཡིན་ནོ་ཞེས་སྨྲས་པ་ན། གདུང་འདེགས་ཀྱི་དོན་བྱེད་ནུས་པ་ལ་ཀ་བའོ་སྙམ་དུ་མིང་དོན་གྱི་འབྲེལ་བ་ཤེས་ནུས་པ་ལྟར་རོ།
For example, when saying "That able to perform the function of load-bearing is the definition or meaning of pillar." to a person whose awareness has already ascertained that able to perform the function of load-bearing but does not know to apply the term "pillar" to it, then they will be able to know the connection between the name and meaning, thinking "That able to perform the function of load-bearing is a pillar."

དེའི་ཕྱིར་མཚན་མཚོན་གཉིས་ལས། མཚན་ཉིད་གོ་སླ་ཞིང༌། དེ་ལ་ལྟོས་ཏེ་མཚོན་བྱ་གོ་དཀའ་བ་ཞིག་དགོས་པ་ཡིན་ནོ། །
Therefore, from among the definition and definiendum: The definition has to be easier to understand and relatively the definiendum has to be more difficult to understand.
