ཡོད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་དང་ཡོད་པ་གཉིས་ཡིན་ཁྱབ་མཉམ་ཡིན་ཏེ། 
**That for which a valid cognition realising its existence exists** and **existent** are mutually pervasive because:
- If it is that for which a valid cognition realising its existence exists, then it is pervaded by being an existent; and
  ཡོད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་ཡིན་ན། ཡོད་པ་ཡིན་པས་ཁྱབ།
- If it is an existent, then it is pervaded by being that for which a valid cognition realising its existence exists.
  ཡོད་པ་ཡིན་ན། ཡོད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་ཡིན་པས་ཁྱབ་པའི་ཕྱིར། 

མེད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་དང་ཡོད་པ་གཉིས་ཡིན་ཁྱབ་མཉམ་ཡིན་ཏེ།
**That for which a valid cognition realising its non-existence does not exist** and **existent** are mutually pervasive because:
- If it is that for which a valid cognition realising its non-existence does not exist, then it is pervaded by being an existent; and
  མེད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་ཡིན་ན། ཡོད་པ་ཡིན་པས་ཁྱབ།
- If it is an existent, then it is pervaded by being that for which a valid cognition realising its non-existence does not exist.
  ཡོད་པ་ཡིན་ན། མེད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་ཡིན་པས་ཁྱབ།

མེད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་དང་མེད་པ་གཉིས་ཡིན་ཁྱབ་མཉམ་ཡིན་ཏེ། 
**That for which a valid cognition realising its non-existence exists** and **non-existent** are mutually pervasive because:
- If it is that for which a valid cognition realising its non-existence exists, then it is pervaded by being a non-existent; and
  མེད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་ཡིན་ན། མེད་པ་ཡིན་པས་ཁྱབ། 
- If it is a non-existent, then it is pervaded by being that for which a valid cognition realising its non-existence exists.
  མེད་པ་ཡིན་ན། མེད་པར་རྟོགས་པའི་ཚད་མ་ཡོད་པ་ཡིན་པས་ཁྱབ་པའི་ཕྱིར།

ཡོད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་དང་མེད་པ་གཉིས་ཡིན་ཁྱབ་མཉམ་ཡིན་ཏེ།
**That for which a valid cognition realising its existence does not exist** and **non-existence** are mutually pervasive because:
- If it is that for which a valid cognition realising its existence does not exist, then it is pervaded by being a non-existent; and
  ཡོད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་ཡིན་ན། མེད་པ་ཡིན་པས་ཁྱབ།

  མེད་པ་ཡིན་ན། ཡོད་པར་རྟོགས་པའི་ཚད་མ་མེད་པ་ཡིན་པས་ཁྱབ།
